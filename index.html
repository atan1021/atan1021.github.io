Banksy

Previous Next

A miniature portable ATM machine that is designed with students in mind. It is meant to be a solution to safely keep money locked away without having the need to pay for ATM fees on campus. Programmed to simulate real banking options such as withdraw, deposit, savings, and more, security measures were added too. Light sensors were used to detect if someone is trying to open the machine without proper access. An alarm would go off if an incorrect pin would be entered more than 3 times in a row. Built using LEGO NXT, simple motors would be used to dispense money.

Face Tracking Module

Previous Next

A Point Grey research camera is used to to follow a face, using a bipolar stepper motor, based off of the direction the subject is looking towards. The camera detects initially detects a face, which then outputs a value to be processed. The algorithm compares current and previous values to determine the direction of the person's head. This is a prototype towards a 2-axis camera on the rear view mirror, which is suppose to help improve face tracking data capture.

Skills used: C++, OpenCV, Raspberry Pi, stepper motor, flyCapture SDK

Stacker

A 2D side stacking game programmed using C on a M3-Cortex ARM processor. Since the platform is only a single core processor, true multitasking is not possible. To smoothly run the game, semaphores were used to prevent tasks from accessing resources that are already allocated. A push-button triggered an interrupt to place the current block, wheres a potentiometer was used to change the difficulty of the game. A joystick allowed for navigation of the games menu.

Skills used: C, ARM Cortex-M3, peripherals, RTOS

Polarity Detecting Robot

Previous Next

A line following robot that detects magnetic objects. Optical encoders were calibrated using infrared LEDs and photo-transistors by mapping the velocity to an oscilloscope. Designed all circuit configurations on the board including trans-impedance amplifiers and Schmidt triggers. All components were soldered on, which were then debugged and tested.

Skills used: soldering, debugging, oscilliscope, signal generators

Gearbox Measuring Tool

Previous Next

A unique measurement device with precision up to a 10th of a millimeter that takes advantage of common gear ratios. A rack and pinion setup was used, where the rack would measure distance relative to its starting position. The gearbox is made entirely out of plywood, which was created from a laser cutter. The sections on the main gears are rasters that help identify the overall displacement.

Skills used: AutoCAD, laser cutting

Cantilever beam

Previous Next

A cantilever beam that is designed to hold as much weight as possible relative to its material weight. Multiple iterations were designed and tested to discover the pros and cons of various designs. The pieces were modeled and assembled in SOLIDWORKS, which were then exported to AutoCAD to be laser cut. Made from balsa wood, the product was able to withstand more than 125 times its own weight before failure.

Skills used: SOLIDWORKS, laser cutting
